Qes: What is the difference between ver,let and const?>

Ans

Var: Var can be updated and re-declared into the scope. It can be declared without initialization

Let: Let can be updated but cannot be re-declared into the scope. It can be declared without initialization.

Const: const cannot be updated or re-declared into the scope. It cannot be declared without initialization.

Qes: What is the difference between arrow function and regular function?

Ans

Arrow function: Inside of a regular JavaScript function, (this) value is dynamic. The regular function can easily construct objects.

Regular function: The behavior of this inside of an arrow function differs considerably from the regular function's this behavior. An arrow function cannot be used as a constructor.
